Über die Mentorendatenbank

Die Mentorendatenbank basiert auf PHP und MySQL sowie HTML, CSS und Javascript (insbesondere jQuery) für die Benutzeroberfläche. Diese wird durch Javascript nur verbessert, das heißt, man kann die Mentorendatenbank auch ohne Javascript-Aktivierung uneingeschränkt nutzen.
